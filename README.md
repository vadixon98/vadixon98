## Featured Project

### [Enzyme Variant Analysis](https://github.com/vadixon98/enzyme-variant-analysis)

Structural and sequence-based characterization of protein enzymes using R/Python via `reticulate`.

| Script | Analysis |
|:-------|:---------|
| Loop Identification & B-Factors | Unstructured regions and per-loop B-factor flexibility from PDB |
| Helix & Sheet Distance Calculations | Cα-to-Cα distances across secondary structure elements |
| Amino Acid Usage by Position & Genus | Conservation and variation by genus from Clustal MSA |

**Stack:** R · Python · BioPython · pandas · numpy · reticulate · PDB
